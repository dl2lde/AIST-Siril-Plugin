# AIST – AstroImage Stretch Tool for Siril

AIST is a modern stretch tool for Siril, designed to provide fast, intuitive and high‑quality stretching for astronomical images.  
It offers a clean interface, real‑time preview, auto‑stretch options and a workflow optimized for deep‑sky processing.

---

## Features
- Real‑time preview inside Siril
- Black / Mid / White point control
- Background and highlight protection
- Enhance slider for faint structures
- Auto White Balance
- Auto Stretch
- Clean and compact UI
- Works directly as a Python plugin

---

## Installation (via Siril Script Manager)

1. Open **Siril**.
2. Go to **Scripts → Script Manager** (or *Manage Script Repositories* depending on version).
3. Add the following repository URL: https://github.com/dl2lde/AIST-Siril-Plugin/ 
4. Click **Update scripts**.
5. The AIST plugin will appear in the **Scripts** menu.
